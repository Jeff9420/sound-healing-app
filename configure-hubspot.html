🧙 HubSpot 配置助手

一步步完成 SoundFlows 的 CRM 集成配置

1

输入你的 Portal ID

📍 获取方式：登录 HubSpot → 点击右上角设置图标 → Account Setup → Account Defaults → 找到 "HubSpot Account ID"

2

输入"资源订阅"表单 GUID

📍 获取方式：HubSpot → Marketing → Forms → 点击你的表单 → 在 URL 中找到（格式: .../forms/12345678/这里就是GUID/edit）
或点击 "Embed" → 在嵌入代码中找到 formId: "GUID"

3

创建"7 日冥想计划"表单

📝 现在去 HubSpot 创建第二个表单
完成后回到这里输入 Form GUID

📋 创建步骤：

  1. HubSpot → Marketing → Forms
  2. 点击 Create form → Embedded form → Blank template
  3. 表单名称: SoundFlows - 7 日冥想计划
  4. 添加以下字段 (按顺序):
    • First name (Single-line text, Required)
    • Last name (Single-line text, Optional)
    • Email (默认已有, Required)
    • + Add new field → Dropdown select
      • Label: Meditation Goal
      • Internal name: meditation_goal
      • Options (每行一个): 
        stress-relief | 压力缓解
better-sleep | 改善睡眠
emotional-balance | 情绪平衡
focus | 专注力提升
general-wellness | 整体健康
      • Required: Yes
    • + Add new field → Dropdown select
      • Label: Preferred Time
      • Internal name: preferred_time
      • Options: 
        morning | 早晨 (6-9 AM)
midday | 中午 (12-2 PM)
evening | 晚间 (6-9 PM)
flexible | 灵活安排
      • Required: Yes
  5. 点击 Options 标签
  6. "What should happen after form submission?" 选择 Display a thank you message
  7. 输入成功消息: ✅ 感谢加入！我们已为你定制了 7 日声音疗愈计划。你将在几分钟内收到欢迎邮件。
  8. 点击右上角 Publish
  9. 复制 Form GUID 并在下方输入
