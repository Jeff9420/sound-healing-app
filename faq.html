如何选择适合我的音频类型？

选择音频类型主要基于您的具体需求和当前状态：

🎯 根据目标选择：

  • 放松和减压 → 雨声、流水声、自然环境音
  • 深度冥想 → 颂钵、脉轮音频、冥想音乐
  • 改善睡眠 → 催眠音频、潜意识疗愈
  • 提高专注力 → 白噪音、专注冥想音乐
  • 情绪调节 → 动物声音、火焰声

⏰ 根据时间选择：

  • 早晨：energizing音频，提升活力
  • 工作时：background音频，提高专注
  • 晚上：relaxing音频，准备睡眠
💡 新手建议：从较短的音频（10-15分钟）开始，逐渐延长使用时间。先尝试雨声或冥想音乐，这些通常最容易适应。
使用声音疗愈有什么注意事项？

⚠️ 重要注意事项：

🔊 音量控制
  • 音量适中，通常为正常音量的50-70%
  • 能听清楚但不刺耳为宜
  • 避免长时间高音量使用
👥 特殊人群
  • 癫痫患者：使用前请咨询医生
  • 孕妇：建议选择温和的自然音效
  • 儿童：需要成人监督，时间不宜过长
🕐 使用时长
  • 首次使用：10-15分钟
  • 适应后：20-30分钟
  • 最长不超过：60分钟
⚡ 出现以下情况请立即停止：头晕、恶心、心跳加速、焦虑加重。声音疗愈是辅助方法，不能替代专业医疗治疗。
为什么有时候音频播放不了？

🔧 常见问题排查：

1. 网络问题
  • 检查网络连接是否稳定
  • 尝试刷新页面
  • 网络较慢时等待缓冲完成
2. 浏览器兼容性
  • 推荐浏览器：Chrome 80+、Firefox 75+、Safari 13+
  • 更新浏览器到最新版本
  • 清除浏览器缓存和Cookie
3. 音频权限
  • 检查浏览器是否阻止了音频播放
  • 点击地址栏中的音频图标，允许音频播放
  • 确保网站没有被静音
4. 设备设置
  • 检查设备音量设置
  • 确保没有启用静音模式
  • 检查耳机或音响连接
🆘 仍然无法解决？尝试：1）重启浏览器；2）使用无痕模式；3）尝试其他设备；4）联系我们的技术支持。

如何获得最佳的声音疗愈体验？

🎧 最佳体验指南：

1. 音频设备
  • 首选：高质量头戴式耳机
  • 次选：立体声音响
  • 避免：手机内置扬声器
2. 环境准备
  • 选择安静、不受打扰的空间
  • 调节到舒适的温度和光线
  • 关闭手机通知和其他干扰源
3. 身体姿势
  • 坐姿：背部挺直，脚平放地面
  • 卧姿：平躺，双臂自然放置身体两侧
  • 避免：过于舒适可能导致睡着的姿势（除非目的是助眠）
4. 心理准备
  • 设定明确的意图（放松、冥想、睡眠等）
  • 专注于当下，放下日常烦恼
  • 保持开放和接受的心态
5. 使用频率
  • 日常维护：每天15-30分钟
  • 深度疗愈：每周2-3次，45-60分钟
  • 应急使用：压力大时随时使用5-10分钟
🎯 专业建议：建立固定的使用时间和地点，让身体形成条件反射。坚持使用比单次长时间使用更有效。
