# 🎨 专业Instagram帖子设计规格

## 📐 技术规格

### 图片尺寸
```
格式：正方形 Instagram Post
尺寸：1080 x 1080 像素
DPI：72
格式：PNG（保持透明度）或JPG（较小文件）
```

### 色彩规格
```
主渐变背景：
- 起始色：#667eea (RGB: 102, 126, 234)
- 结束色：#764ba2 (RGB: 118, 75, 162)
- 渐变方向：从上到下

文字颜色：
- 主文字：#FFFFFF (纯白色)
- 辅助文字：#F8F9FA (浅灰白色)
```

## 🎯 视觉层次设计

### 布局结构（从上到下）
```
1. 顶部空白区域 (100px)
2. 主标题区域 (150px)
   "💡 MEDITATION TIP MONDAY"
3. 内容区域 (600px)
   核心文字内容
4. 品牌区域 (80px)
   "SoundFlows.app - Free Sound Therapy"
5. 底部空白区域 (150px)
```

### 字体规格
```
主标题：
- 字体：Montserrat Bold 或 Roboto Bold
- 大小：48px
- 行距：1.2
- 对齐：居中

正文内容：
- 字体：Open Sans Regular 或 Roboto Regular
- 大小：32px
- 行距：1.4
- 对齐：居中

品牌信息：
- 字体：Roboto Light
- 大小：24px
- 对齐：居中
```

## 🛠️ Canva制作步骤

### 第1步：创建基础画布
```
1. 打开Canva
2. 点击"创建设计"
3. 选择"Instagram帖子" (1080x1080)
4. 选择空白模板
```

### 第2步：设置渐变背景
```
1. 点击"背景"
2. 选择"渐变"
3. 设置双色渐变：
   - 颜色1：#667eea
   - 颜色2：#764ba2
   - 方向：垂直（从上到下）
```

### 第3步：添加主标题
```
1. 点击"文字"
2. 选择"添加标题"
3. 输入："💡 MEDITATION TIP MONDAY"
4. 设置格式：
   - 字体：Montserrat Bold
   - 大小：48
   - 颜色：白色 (#FFFFFF)
   - 对齐：居中
   - 位置：距离顶部200px
```

### 第4步：添加正文内容
```
1. 点击"文字"
2. 选择"添加正文"
3. 输入正文内容
4. 设置格式：
   - 字体：Open Sans Regular
   - 大小：32
   - 颜色：白色 (#FFFFFF)
   - 行距：1.4
   - 对齐：居中
   - 位置：居中对齐
```

### 第5步：添加品牌信息
```
1. 点击"文字"
2. 选择"添加正文"
3. 输入："SoundFlows.app - Free Sound Therapy"
4. 设置格式：
   - 字体：Roboto Light
   - 大小：24
   - 颜色：白色 (#FFFFFF)
   - 对齐：居中
   - 位置：距离底部100px
```

### 第6步：下载和保存
```
1. 点击右上角"下载"
2. 选择格式：PNG（推荐）
3. 点击"下载"
4. 保存到手机相册
```
