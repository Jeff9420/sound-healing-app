步骤1：注册Mailgun账户

请按照以下步骤完成注册：

  • 访问 https://www.mailgun.com
  • 点击 "Sign Up" 或 "Get Started for Free"
  • 填写注册信息：
    • 邮箱：您的有效邮箱地址
    • 密码：创建安全密码
    • 公司：SoundFlows（可填个人名称）
  • 验证邮箱：查收邮件并点击验证链接
提示：注册后会自动分配一个sandbox域名，格式如 sandbox12345.mailgun.org
2

步骤2：获取API密钥

登录后按以下步骤获取API密钥：

  • 进入Dashboard仪表板
  • 左侧菜单点击 "Sending" → "Domain Settings"
  • 找到您的sandbox域名（如 sandbox12345.mailgun.org）
  • 点击域名进入详情页
  • 找到 "API Keys" 或 "Private API Key"
  • 点击 "Show" 并复制密钥（格式：key-xxxxxxxxxxxx）
重要：请保存好您的API密钥，它不会再次显示
3

步骤3：配置到网站

请在下方填入您的Mailgun配置信息：

// 将此配置添加到 assets/js/email-service-mailgun.js
const MAILGUN_CONFIG = {
    apiKey: 'YOUR_API_KEY_HERE',
    domain: 'YOUR_DOMAIN_HERE'
};
