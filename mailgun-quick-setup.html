  4. 完成注册后自动进入Dashboard
2

🔑 获取API密钥（1分钟）

登录后按以下步骤获取：

  1. 左侧菜单点击 "Sending" → "Domain Settings"
  2. 找到您的sandbox域名（如 sandbox12345.mailgun.org）
  3. 点击域名进入详情页
  4. 找到 "API Keys" 或 "Private API Key"
  5. 点击 "Show" 并复制密钥
