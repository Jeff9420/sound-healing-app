## 🚀 AdSense申请步骤

### 第一步：创建AdSense账户

1. 访问 [Google AdSense](https://www.google.com/adsense/)
2. 点击"开始使用"
3. 选择"我想通过我的网站获利"
4. 输入网站URL: `https://soundflows.app`
5. 选择国家/地区
6. 选择付款币种

### 第二步：网站关联

1. 将提供的AdSense代码添加到网站
2. 验证网站所有权

1. 提供身份验证信息

3. 配置付款方式

## 🛠️ 技术集成步骤

### 1. 添加AdSense代码

将以下代码添加到 `<head>` 标签中（已在index.html中预留位置）：

```html
<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-YOUR_PUBLISHER_ID"
     crossorigin="anonymous"></script>
```

### 2. 更新AdManager配置

在 `assets/js/ad-manager.js` 中更新您的Publisher ID：

```javascript
// 替换 YOUR_PUBLISHER_ID 为您的实际ID
script.src = 'https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-YOUR_ACTUAL_ID';
```

### 3. 创建广告单元

在AdSense后台创建以下广告单元：

#### 主页顶部原生广告
- **类型**: 原生广告
- **尺寸**: 响应式
- **位置**: 主页标题下方

#### 轮播下方展示广告
- **类型**: 展示广告
- **尺寸**: 300x250 (中等矩形)
- **位置**: 3D轮播下方

### 4. 配置广告单元ID

更新 `ad-manager.js` 中的广告单元ID：

```javascript
adElement.setAttribute('data-ad-client', 'ca-pub-YOUR_PUBLISHER_ID');
adElement.setAttribute('data-ad-slot', 'YOUR_AD_SLOT_ID');
```
