📧 Formspree 完整教程

一步步教您如何创建Formspree表单并获取Form ID

1

访问Formspree官网

首先，打开浏览器访问Formspree官网。这是创建表单的第一步。

🔗 点击访问 formspree.io

Formspree首页 - 点击"Sign Up"或"Get Started"

[网站截图：Formspree首页]
2

注册账号

Formspree只需要邮箱验证，不需要手机！填写您的邮箱地址即可。

💡 注册提示

  • 使用您常用的邮箱（Gmail、QQ邮箱、163邮箱等）
  • 密码建议至少8位，包含字母和数字
  • 查收验证邮件并点击验证链接

注册表单 - 只需填写邮箱和密码

[网站截图：注册表单]
3

创建新表单

登录后，点击"New Form"创建您的第一个表单。

Dashboard页面 - 点击"New Form"按钮

[网站截图：Dashboard页面]
4

配置表单信息

填写表单的基本信息，最重要的部分是设置接收邮件的邮箱地址。

📝 表单配置示例

✅ 这就是接收表单提交的邮箱地址

表单设置页面

[网站截图：表单设置]
5

获取您的Form ID

创建表单后，您会看到表单的详情页面。Form ID就在这里！

📍 Form ID 位置

在表单详情页面的顶部，您会看到类似这样的URL：

https://formspree.io/f/mzbqknyp

加粗的部分就是您的Form ID！

📌 Form ID 是什么？

Form ID是每个表单的唯一标识符，用于将表单提交连接到您的Formspree账号。它通常是：

  • 8-10个字符长
  • 由字母和数字组成
  • 格式：mzbqknyp, xqjpljko 等

Form ID 在页面上的显示位置

[网站截图：Form ID位置]
6

在您的网站中使用

获取Form ID后，您可以在网站中使用了！

<!-- HTML表单代码 -->
<form action="https://formspree.io/f/您的FormID" method="POST">
  <label>您的邮箱：</label>
  <input type="email" name="email" required>

  <label>留言内容：</label>
  <textarea name="message" required></textarea>

  <button type="submit">发送</button>
</form>

<!-- JavaScript代码 -->
<script>
// 保存Form ID到localStorage
localStorage.setItem('formspreeFormId', '您的FormID');
</script>
⬇️

📝 实际示例

如果您的Form ID是 mzbqknyp，那么代码应该是：

<form action="https://formspree.io/f/mzbqknyp" method="POST">
  <input type="email" name="email" placeholder="您的邮箱">
  <textarea name="message" placeholder="您的留言"></textarea>
  <button type="submit">发送</button>
</form>

⚠️ 重要提示

  • 保存Form ID：请将Form ID保存好，这是您表单的唯一标识
  • 免费额度：免费版每月可以接收50封邮件
  • 测试方法：提交表单后，登录Formspree查看Submissions
  • 自定义：可以在Formspree后台设置自动回复等高级功能
